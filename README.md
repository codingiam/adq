# A Drogon's Quest

## Requirements

- Ruby 2.6.3+
- Terminal that supports ANSI/VT100

## Usage

```bash
$ ruby adq.portable.min.rb
```

Keys:

- W - Up
- S - Down
- A - Left
- D - Right
- Q - Quit

Symbols:

- \* - Drogon
- \+ - Goat
- \. - Ground
